# KT
KT is a multipurpose discord bot.

## Features
- Voice responses for players joining channel
- Break messages that can be sent to a channel at a set interval
- Custom commands

## Commands
- `*hello` - Says hello
- `*toggle_voice_settings` - Toggles voice responses
